$\lim\limits_{x \to 0} \frac {f\left [ g\left ( x \right )  \right ] - f\left [ h\left ( x \right )  \right ]}{x^k} \overset{拉}{=}\lim\limits_{x \to 0}\frac{g\left ( x \right ) - h\left ( x \right ) }{x^k}{f'\left ( \xi\right )}， \\其中\xi介于g\left (  x\right ) 与f\left ( x \right )之间$
1. 若$x\rightarrow 0$时$f'\left( \xi \right)$是非零因子，则拉格朗日一定可以求出该极限
2. 若$x\rightarrow 0$时$f'\left( \xi \right)$是零因子，且能确定出$f'\left( \xi \right)$关于x的等价无穷小，则拉格朗日一定可以求出该极限
3. 若$x\rightarrow 0$时$f'\left( \xi \right)$是零因子，且不能确定出$f'\left( \xi \right)$关于x的等价无穷小，则拉格朗日不一定可以求出该极限，要具体问题具体分析
4. 若$f'\left( \xi \right)$不是因子，则不能用拉格朗日去求极限！如$\lim\limits_{x \to 0} \frac {f\left [ g\left ( x \right )  \right ] - f\left [ h\left ( x \right )  \right ] - x^2}{x^3}$

例1 $\lim\limits_{x \to 0} \frac{\left ( e - e^{\cos x} \right )}{x^2}$ 
=$\lim\limits_{x \to 0}\frac{1-\cos x}{x^2}e^\xi$，其中$\xi$介于$1$与$\cos x$之间，不能写$\xi\in \left(1,\cos x \right)$之间，因为不知道$\cos x$与$1$的大小关系
由于$\lim\limits_{x \to 0}\cos x=1$，而$\xi$在$1$与$\cos x$之间，故由夹逼准则，$\lim\limits_{\xi \to 0}\xi=1$
原极限=$\lim\limits_{x \to 0}\frac{\frac{1}{2}x^2}{x^2}e^\xi=\lim\limits_{x \to 0}\frac{1}{2}e^\xi=\frac{1}{2}$

例2 $\lim\limits_{x \to 0} \frac{\left ( e^x - e^{\sin  x} \right )}{x^3}$ 
= $\lim\limits_{x \to 0}\frac{x-\sin x}{x^3}e^{\xi}$，其中$\xi$介于$x$与$\sin x$之间 由于 $\lim\limits_{x \to 0}x=\lim\limits_{x \to 0}\sin x=0$
所以$\lim\limits_{\xi \to 0}\xi=0$
=$\lim\limits_{x \to 0}\frac{1}{6}e^{\xi}=\frac{1}{6}$

例3 $\lim\limits_{x \to 0}\frac{\cos x-\cos(sinx)}{x^4}$
$=\lim\limits_{x \to 0}\frac{x-\sin x}{x^4}\left(-\sin x \right)\xi$，其中$\xi$介于$x$与$\sin x$之间，易得$\lim\limits_{x \to 0}\xi=0$
$=-\frac{1}{6}\lim\limits_{x \to 0}\frac{\sin \xi}{x}$
但之前知道$\lim\limits_{x \to 0}\xi=0$无法解出这题因为$x$的极限也是$0$，分子除以分母就是零比零型，是未定式，无法计算
这里需要更精确的夹逼准则：$\xi$介于$x$与$\sin x$之间，又由于$x\rightarrow0$时$\sin x \sim x$，故$x\rightarrow0$时$\xi\sim x$，因为放大之后的函数与缩小之后的函数是等价无穷小，所以夹在它们中间的一定与他们都等价
可以这么理解：$B$夹在$A$和$C$中间，而$A\sim C$所以$B\sim C，B\sim A$
原极限$=-\frac{1}{6}\lim\limits_{x \to 0}\frac{\xi}{x}=-\frac{1}{6}\lim\limits_{x \to 0}\frac{x}{x}=\frac{1}{6}$

例4 $\lim\limits_{x \to 0}\frac{\cos 2x-\cos x}{x^2}$
$=\lim\limits_{x \to 0}\frac{\left(2x- x\right)\left(-\sin \xi\right)}{x^2}$，其中$\xi$介于$2x$与$x$之间
$=\lim\limits_{x \to 0}\frac{-\sin \xi}{x}$
$\lim\limits_{x \to 0}2x=0，\lim\limits_{x \to 0}x=0$，所以$\lim\limits_{x \to 0}\xi=0$
$=\lim\limits_{x \to 0}\frac{\xi}{x}$
与上题不同，$x\rightarrow0$时，$2x$与$x$不等价，所以无法知道$\xi$与$x$的大小关系，本题用拉格朗日无法求出，这就是上面的第三种情况
另解：添项减项
=$\lim\limits_{x \to 0}\frac{\left(\cos 2x-1\right)-\left(\cos x-1\right)}{x^2}=-\frac{3}{2}$

例5 $\lim\limits_{x \to 0}\frac{\cos 3x-\cos x}{x}$
 $=\lim\limits_{x \to 0}\frac{\left(3x-x\right)\left( -\sin \xi\right)}{x}$，其中$\xi$在3x和x之间，可得$\lim\limits_{\xi \to 0}\sin \xi=0$
 $=-2\lim\limits_{x \to 0}\sin \xi=-2\lim\limits_{x \to 0}\xi=0$
 例6 设$f\left(x\right)$在$\left[ -1,1 \right]$二阶可导，$f'\left( 0 \right)=0，f''\left( 0 \right)=4$，求$\lim\limits_{x \to 0}\frac{f\left(x\right)-f\left[\ln\left(1+x\right)\right]}{x^3}$
 $=\lim\limits_{x \to 0}\frac{\left[x-\ln\left(1+x\right)\right]f'\left(\xi \right)}{x^3}=\frac{1}{2}\lim\limits_{x \to 0}\frac{f'\left(\xi \right)}{x}$
 其中$\xi$在$x$和$\ln\left(1+x\right)$之间，
 易得$\lim\limits_{x \to 0}f\left(\xi\right)=0，$且由$x$与$\ln\left(1+x\right)$等价可知，当$x\rightarrow 0$时，$f\left(\xi\right)\sim x\sim\ln\left(1+x\right)$
 到这里发现题目中还有$f'\left( 0 \right)=0，f''\left( 0 \right)=4$没有用上，发现有$f'\left( 0 \right)=0$，于是可以：
 $\frac{1}{2}\lim\limits_{x \to 0}\frac{f'\left(\xi \right)}{x}=\frac{1}{2}\lim\limits_{x \to 0}\frac{f'\left(\xi \right)-f'\left( 0 \right)}{x}=\frac{1}{2}\lim\limits_{x \to 0}\frac{f'\left(\xi \right)-f'\left( 0 \right)}{\xi-0}\frac{\xi}{x}=\frac{1}{2}\lim\limits_{x \to 0}f''\left( \xi\right)\frac{\xi}{x}=2$

例7$\lim\limits_{x \to 0} \int_{x}^{sinx}\frac{ln^2\left ( t+\sqrt{1+t^2}  \right )}{x^n}dt =a\left ( a\ne0 \right )$求$n$和$a$
不能洛必达，因为洛必达是后验证法则无法判断结果极限是否存在
由积分中值定理：
=$\lim\limits_{x \to 0} \frac{\left(\sin x-x\right)ln^2\left ( \xi+\sqrt{1+\xi^2}  \right )}{x^n}$，$\xi$介于$\sin x$与$x$之间，易得$\xi\sim x$
=$\lim\limits_{x \to 0} \frac{\left(\sin x-x\right)x^2}{x^n}=\lim\limits_{x \to 0} \frac{\left(-\frac{1}{6}x^3\right)x^2}{x^n}=-\frac{1}{6}\lim\limits_{x \to 0} \frac{1}{x^{n-5}}$
由题$-\frac{1}{6}\lim\limits_{x \to 0} \frac{1}{x^{n-5}}=a$
故$n=5$
