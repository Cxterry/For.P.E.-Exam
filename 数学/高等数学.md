# 3 积分
区间再现
$\int_{a}^{b}f\left ( x \right )dx= \int_{a}^{b}f\left ( a+b-x \right )dx$
区间再现在对称区间推论：
$\int_{-a}^{a}f\left ( x \right )dx= \int_{-a}^{a}f\left (-x \right )dx=\int_{-a}^{a}\frac{1}{2} \left[ f\left ( x \right )+f\left ( -x \right ) \right]dx$
显然，$f\left ( x \right )+f\left ( -x \right )$是偶函数
则$\int_{-a}^{a}f\left ( x \right )dx=2\int_{0}^{a}f\left ( x \right )+f\left ( -x \right )dx$
